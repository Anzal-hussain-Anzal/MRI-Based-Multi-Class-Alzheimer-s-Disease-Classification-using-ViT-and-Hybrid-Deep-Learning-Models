# MRI-Based-Multi-Class-Alzheimer-s-Disease-Classification-using-ViT-and-Hybrid-Deep-Learning-Models
MRI-Based Alzheimer’s classification using Vision Transformers (ViT) and CNNs (InceptionV3, ResNet50, Xception). Implements a hybrid ensemble for high accuracy (96.3%) on multi-class MRI data. Includes preprocessing, training, evaluation, and reproducible code for early Alzheimer’s detection
